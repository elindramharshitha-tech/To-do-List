# 📝 To-Do List App

## 📌 Project Description

This is a simple To-Do List application built using **HTML, CSS, and JavaScript**.
It allows users to manage their daily tasks by adding, viewing, and deleting items.


## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* 💾 Data stored in browser (LocalStorage)
* 🔄 Tasks remain after page refresh



## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)



## 📁 Project Structure

* `index.html` → Main UI
* `style.css` → Styling
* `script.js` → Logic


## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser



## 🎯 Purpose of the Project

This project is created to practice:

* DOM Manipulation
* Event Handling
* LocalStorage usage
* Frontend development basics
